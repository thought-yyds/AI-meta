# AI Agent Assistant - 功能说明

## 🎯 核心功能

### 1. 三栏布局设计

- **左侧栏：Agent 列表**
  - 显示所有可用的 Agent
  - 支持创建新 Agent
  - 支持删除 Agent
  - 点击选择当前使用的 Agent

- **中间主体区：任务面板 + 聊天面板**
  - 任务执行可视化（Mermaid 流程图）
  - 实时聊天界面
  - 任务操作按钮（重试/跳过/修改参数/继续）

- **右侧栏：任务洞察面板（可折叠）**
  - 任务概览统计
  - 当前步骤详情
  - 步骤历史记录
  - 性能指标

### 2. 任务执行可视化

使用 Mermaid 流程图实时展示任务执行流程：

```
用户意图解析 → 任务规划 → 工具选择 → 工具执行 → 子任务执行 → 汇总结果 → 输出
```

**状态颜色**：
- 🟢 已完成（绿色）
- 🔵 运行中（蓝色，带脉冲动画）
- 🔴 失败（红色）
- 🟠 已跳过（橙色）
- ⚪ 待执行（灰色）

### 3. 任务操作功能

当任务执行时，提供以下操作按钮：

- **🔁 Retry Step** - 重试当前步骤
- **⏭ Skip Step** - 跳过当前步骤
- **➕ Modify Tool Args** - 修改工具参数（JSON 格式）
- **▶ Continue** - 继续执行任务

### 4. 任务洞察面板

**任务概览**：
- 总步骤数
- 已完成步骤数
- 失败步骤数
- 已用时间

**当前步骤详情**：
- 步骤名称
- 执行状态
- 使用的工具
- 工具参数
- 执行结果
- 错误信息（如有）

**步骤历史**：
- 所有步骤的执行记录
- 每个步骤的状态和工具信息

**性能指标**：
- 平均步骤执行时间
- 成功率
- 重试次数

## 🚀 使用说明

1. **选择 Agent**：在左侧栏选择一个 Agent，或创建新 Agent
2. **发送消息**：在中间聊天面板输入消息并发送
3. **查看执行流程**：任务执行时，会自动显示流程图
4. **操作任务**：如果任务失败或需要调整，使用操作按钮
5. **查看洞察**：在右侧面板查看详细的任务执行信息

## 📦 技术栈

- **Vue 3** - 前端框架
- **TypeScript** - 类型安全
- **Mermaid** - 流程图可视化
- **Vite** - 构建工具

## 🎨 设计特点

- 现代化 UI 设计
- 渐变色彩主题
- 毛玻璃效果
- 流畅的动画过渡
- 响应式布局
- 实时状态更新

## 📝 注意事项

1. Mermaid 流程图需要浏览器支持 SVG
2. 任务执行状态会实时同步到右侧洞察面板
3. 修改工具参数需要提供有效的 JSON 格式
4. 任务可以随时暂停、重试或跳过
